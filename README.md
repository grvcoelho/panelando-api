# panelando-api

Panelando RESTful api

## Getting Started

You need [docker](https://docs.docker.com/engine/installation/) installed for these instructions!

1. Build the application

```sh
$ docker-compose build
```

2. Run the application

```sh
$ docker-compose up
```
